Wait What? Hello World??!

This site may or may not assume things about what you know. It is your choice to collaborate and ask questions if anything is unclear. The goal of this site is to create a hub for sorting out technical solutions, what technologies to use, guides, and whatever else to aid in the competitive environment known as the hackathon. Please contribute!

Popular Technologies

Languages

  • Ruby: http://ruby.learncodethehardway.org
  • Python: http://learnpythonthehardway.org

Web Development Framework

  • Ruby on Rails (if you like Ruby) http://rubyonrails.org
  • Django (if you like Python) https://www.djangoproject.com
  • Sinatra, because it's just easier for lighter stuff http://www.sinatrarb.com

Hosting

  • Pick one, any one, such as Heroku, Dreamhost, etc. (should work with your web framework of choice)

Databases

  • SQLite3, which is to say relational databases, in general
  • MySQL, or Postgres, pick one, or both
  • Non-relational: MongoDB, CouchDB, Riak, etc

Tools

  • the command line, get familiar with BASH
  • tmux, allows persistent connections to remote and local processes, and allows sharing (collaboration)
  • Git, no exceptions http://git-scm.com/book
  • GitHub, also very important
  • pick an editor and get good at it: Emacs, Vim, Sublime Text, etc.

Other Web Topics

  • CSS in general, and CSS3
  • Javascript - The Good Parts http://www.youtube.com/watch?v=hQVTIJBZook
  • Underscore.js, or other JS utility library http://underscorejs.org
  • jQuery http://jquery.com
  • D3.js, if interested in data visualizations http://d3js.org
  • Foundation http://foundation.zurb.com
  • Bootstrap http://twitter.github.com/bootstrap/
